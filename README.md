# DankDash Backend

Premium Cannabis Delivery Platform - Backend API

## Features
- Flask REST API
- PostgreSQL database
- User authentication
- Order management
- Admin panel
- Email integration (SendGrid)
- SMS integration (Twilio)

## Setup
```bash
pip install -r requirements.txt
python src/main.py
```

## Deploy to Railway
```bash
railway up
```
